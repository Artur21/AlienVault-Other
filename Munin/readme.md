Munin EPS
=====================

Add EPS stats to Munin

Installation
-------

*Copy the contents of the script attached to /etc/munin/plugins/
*Name the file ossim_agent
*chmod a+x ossim_agent
*restart munin-node: /etc/init.d/munin-node restart
*Done!
